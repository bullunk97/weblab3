Project Panel Otomatisasi Penerangan

Project ini merupakan penerapan dari modul instalasi penerangan dengan penggunaan sensor dan sistem otomatis. Panel ini dirancang untuk menyalakan lampu secara otomatis berdasarkan intensitas cahaya dan waktu. Mahasiswa belajar merancang rangkaian, mengukur daya, serta memprogram kontroler sederhana.

Komponen: sensor LDR, timer relay, kontaktor, dan MCB. Project ini dikembangkan oleh mahasiswa semester 4.
